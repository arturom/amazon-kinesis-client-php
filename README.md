# Amazon Kinesis Client for PHP

This package provides a PHP interface to the Amazon Kinesis Client Library (KCL) MultiLangDaemon,
which is part of the [Amazon Kinesis Client Library for Java](https://github.com/awslabs/amazon-kinesis-client) project. The code was ported from Python to PHP based on the [Amazon Kinesis Client Library for Python repo](https://github.com/awslabs/amazon-kinesis-client-python).
